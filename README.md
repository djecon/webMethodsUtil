"# webMethodsUtil" 
getOsInfo is a Java code example of how to obtain some useful information from the OS etc. This utility can be used to extract the information like:
systemLoadAverage
JVMmaxMemory
JVMUsedMemory
JVMFreeMemory
JVMStartDate
JVMUptimeSeconds
currentThreadCount
peakThreadCount
totalDiskSpace
freeDiskSpace
